# Tunneling Correction Calculation

The tunneling correction factor $\kappa$ was determined using the Wigner formula:

$$\kappa = 1 + \frac{1}{24} \left( \frac{h \nu^{\ddagger}}{k_B T} \right)^2$$

### Parameters:
- $h$: Planck constant
- $k_B$: Boltzmann constant 
- $T$: Temperature (300 K)
- $\nu^{\ddagger}$: Absolute value of the imaginary frequency
